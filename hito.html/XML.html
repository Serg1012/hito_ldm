Historia de XML

XML (eXtensible Markup Language) fue desarrollado por el World Wide Web Consortium (W3C) en 1996 como una versión simplificada y más flexible del lenguaje de marcado SGML (Standard Generalized Markup Language). Su objetivo principal era facilitar el intercambio de datos estructurados en la web.

Normas de Uso y Sintaxis de XML

XML utiliza etiquetas para definir la estructura y el significado de los datos. Las etiquetas están organizadas en una estructura jerárquica de árbol, donde cada elemento puede contener otros elementos y atributos.

Algunas normas de uso comunes de XML incluyen:

  • DTD (Document Type Definition): Define la estructura y los tipos de datos permitidos en un documento XML.
  • XSD (XML Schema Definition): Ofrece una forma más poderosa y flexible de definir la estructura y la validación de datos en un documento XML.
  • XPath: Permite realizar consultas en documentos XML para extraer información específica.
  • XSLT (XML Stylesheet Language Transformations): Permite transformar documentos XML en otros formatos, como HTML o XML.
